## 💾 설치 방법

### 필요 조건
- PC (Windows) Steam 버전
- 별도 프로그램 설치 불필요 (BepInEx 포함)

### 설치 순서

**1단계 — 다운로드**

[GitHub Releases](https://github.com/hanpaemo/minimalist-tower-defense-korean-patch/releases)에서 최신 ZIP 파일을 다운로드합니다.

**2단계 — 압축 해제**

게임 설치 폴더에 **덮어쓰기**로 압축을 해제합니다.

```
기본 경로: C:\Program Files (x86)\Steam\steamapps\common\Minimalist Tower Defense
```

**3단계 — 게임 실행**

Steam에서 정상 실행하면 자동으로 한국어가 적용됩니다.

> ⚠️ 첫 실행 시 BepInEx 초기화로 로딩이 약간 길어질 수 있습니다. 정상입니다.

## ❓ 자주 묻는 질문

**Q. 게임이 여전히 영어로 표시돼요**

A. 게임을 완전히 종료한 뒤 다시 실행해 보세요. 패치가 I2 Localization 언어 설정을 자동으로 한국어로 전환합니다.

**Q. 일부 특수 방어탑 이름이 영어예요**

A. 이벤트/히든 방어탑까지 전부 번역되어 있습니다. 영어로 보이는 항목이 있다면 [Issues](https://github.com/hanpaemo/minimalist-tower-defense-korean-patch/issues)에 알려주세요.

**Q. 게임 업데이트 후 번역이 안 돼요**

A. 게임 업데이트 시 패치를 다시 설치해야 할 수 있습니다. Releases에서 최신 버전을 확인해주세요.

**Q. Steam 파일 무결성 검사 후 번역이 사라졌어요**

A. 무결성 검사는 패치 파일을 삭제하므로, 검사 후 패치를 다시 설치하시면 됩니다.
